# New Year site 
## About 
A festive, interactive website with a countdown to the next New Year, falling snow animation, and a “Letter to Santa” form powered by Formspree. 
Built as a learning project using **HTML**, **CSS**, and **JavaScript**.

## Features

- **Live countdown timer** to the next New Year
- **Animated falling snowflakes** using CSS keyframes
- **Navigation bar** with links to different sections/pages
- **Interactive “Letter to Santa” form** submitting via Formspree
- Fully **responsive background** and festive styling
- No backend required — purely static site
